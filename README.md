# ファッションサイト　-- さまざまな種類のアイテムを扱う、アパレルブランドの商品を販売するwebサイト

**1.サービス概要**
　こちらのwebサイトは、アパレル商品を扱う会社様向けのwebサイトになります。アパレルの商品の一覧と、それぞれのジャンルに沿った商品の一覧を選択できます。
